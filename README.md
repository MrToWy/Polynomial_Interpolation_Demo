# Polynomial_Interpolation_Demo

Abrufbar unter [CG3.Töbi.de](https://cg3.töbi.de)


## Installation
```
npm install
```

## Start Dev-Server
```
npm run start
```

## Build production sources
```
npm run build
```

## Additional libraries
- mathjs.org
- threejs.org
- latex.js.org
